## RRE v0.5.1 — AWS cloud cutover (canonical)

Canonical release on `main` after the AWS pivot: Stage B networking, Bedrock Quick Review, Amazon Transcribe, slim cloud image, and deploy/pause ops on the default branch.

### Highlights

**AWS substrate (v0.5.0 era → landed here)**
- ECS Fargate (API + UI), ALB, RDS PostgreSQL, ECR, Secrets Manager, CloudWatch
- Stage B no-egress: VPC endpoints (Bedrock, Transcribe, ECR, S3, Secrets, Logs, STS)
- GitHub Actions OIDC deploy (`dev-github-deploy`)
- `/api/live` for ALB; AWS-3f post-deploy smoke

**LLM / ASR**
- `LLMProvider` + Amazon Bedrock (Converse) for modules on AWS
- `TranscriptionProvider` + Amazon Transcribe (S3 job → output JSON in-bucket)
- Local path unchanged: Whisper + optional pyannote (`TRANSCRIPTION_PROVIDER=whisper`, `.[local]`)

**Slim cloud image**
- `Dockerfile.cloud` — no torch / Whisper / pyannote
- Streamlit only in `.[ui]` / `.[dev]` (not in cloud API image)
- Fargate API **1024 CPU / 2048 MiB** (512/2048 failed health during cutover)
- UI **256 / 1024** with longer health startPeriod; deployment circuit breaker + `minHealthyPercent=100`

**Ops**
- Deploy on push to `main` when runtime/infra paths change (docs-only stays quiet) + `workflow_dispatch`
- **Pause AWS dev** workflow: ECS desired 0 + stop RDS

### Breaking / operator notes

- AWS default: `LLM_PROVIDER=bedrock`, `TRANSCRIPTION_PROVIDER=transcribe`, diarization off
- Local install for UI/tests: `pip install -e ".[dev]"` (includes Streamlit); audio: add `.[local]`
- After idle sessions: Actions → **Pause AWS dev**
