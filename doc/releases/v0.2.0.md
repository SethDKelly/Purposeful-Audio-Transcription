## RRE MVP release

First complete Relationship Reasoning Engine MVP (phases A-G).

### Highlights
- Transcript ingestion with evidence indexing and quote IDs
- 5 analysis modules + meta-synthesis
- Workflows: `quick_review`, `full_mvp`
- Synthesis engine with convergence/divergence
- Streamlit report UI with exports
- Testing hardening: golden/red-team fixtures, safety validation, evidence summarization
