# v0.9.0 — Cases and Longitudinal Analysis

**Date:** July 2026  
**Theme:** Multi-session cases, longitudinal comparison/synthesis, report packages, and analyst finding feedback.

## Highlights

| Area | Delivered |
|------|-----------|
| **Cases** | `cases` table; transcript `case_id` / `session_label` / `session_date`; CRUD + assign API |
| **Case dashboard** | Streamlit Cases expander: create, assign, list sessions |
| **Longitudinal compare** | `POST /api/exploration/compare-transcripts` (earliest vs latest themes) |
| **Longitudinal synthesis** | `longitudinal_synthesis` module + `POST /api/cases/{id}/longitudinal-synthesis` |
| **Report package** | ZIP export: manifest + report + evidence appendix + findings index |
| **Finding feedback** | `finding_feedback` ratings (helpful / unhelpful / unsure) + API + finding-card UI |
| **Migrations** | Alembic `008`–`009` |
