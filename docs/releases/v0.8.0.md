# v0.8.0 — Structured Persistence and Graph Reasoning

**Date:** July 2026  
**Theme:** Promote findings, constructs, and relationships from JSON blobs into first-class persistent objects; merge, score, explore, and synthesize over that structure.

## Highlights

| Area | Delivered |
|------|-----------|
| **Normalized findings** | `findings` + evidence quotes + alternative explanations; written on module completion; `parsed_output` retained for audit |
| **Normalized constructs** | `constructs` + evidence + sources; ontology resolve/warn |
| **Relationships** | `construct_relationships` with ontology + dangling-link warnings |
| **Graph merge** | Cross-module similarity merge; sources/evidence preserved on canonical nodes |
| **Convergence scoring** | Deterministic strong / moderate / weak / contested with rationale JSON |
| **Exploration** | `GET /api/workflow-runs/{id}/structured-graph` + Streamlit **Structured inventory** tables |
| **Synthesis** | Meta-synthesis handoff prefers structured inventory; merge/score run before synthesis |
| **Migrations** | Alembic `005`–`007` |
